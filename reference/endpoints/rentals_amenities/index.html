Rentals amenities

  1. List rentals amenities
  2. Get a single rentals amenity

List rentals amenities

List all rentals amenities for given account.

GET /rentals_amenities
  • OAuth Scopes:
  • public

Response

{
  "links": {
    "rentals_amenities.amenity": "https://bookingsync.dev/api/v3/amenities/{rentals_amenities.amenity}",
    "rentals_amenities.rental": "https://bookingsync.dev/api/v3/rentals/{rentals_amenities.rental}"
  },
  "rentals_amenities": [
    {
      "links": {
        "amenity": 82,
        "rental": 1
      },
      "id": 3,
      "created_at": "2014-08-23T17:44:09Z",
      "updated_at": "2014-08-23T17:44:09Z"
    }
  ]
}

Get a single rentals amenity
