Billing Addresses

  1. List billing addresses
  2. Get a single billing address
  • OAuth Scopes:
  • payments_read / payments_write

Parameters

Name Type Read/Write Description
id Integer Read Billing Address’s id.
address1 String Read Billing Address’s first address.
address2 String Read Billing Address’s second address.
city String Read Billing Address’s city.
country_code String Read Billing Address’s country_code, list of codes is available at wikipedia.
state String Read Billing Address’s state.
zip String Read Billing Address’s zip.
created_at Time Read Billing Address’s create time.
updated_at Time Read Billing Address’s update time.

List billing addresses

List all billing addresses for given account.

GET /billing_addresss
  • OAuth Scopes:
  • payments_read / payments_write

Response

{
  "links": {
    "billing_addresses.payment": "https://www.bookingsync.com/api/v3/payments/{billing_addresses.payment}"
  },
  "billing_addresses": [
    {
      "id": 12,
      "address1": "Bernhard Creek",
      "address2": null,
      "city": "East Winifred",
      "country_code": "US",
      "state": "Monatana",
      "zip": "07671-8254",
      "created_at": "2014-10-20T17:05:27Z",
      "updated_at": "2014-10-20T17:05:27Z",
      "links": {
        "payment": 33
      }
    }
  ]
}

Get a single billing address

Returns a single billing address identified by ID.

GET /billing_addresses/:billing_address_id
  • OAuth Scopes:
  • payments_read / payments_write
