List sources

List all sources for given account.

GET /sources
  • OAuth Scopes:
  • bookings_read / bookings_write

Response

{
  "links": {
    "sources.account": "https://test.host/api/v3/accounts/{sources.account}"
  },
  "sources": [
    {
      "links": {
        "account": 1
      },
      "id": 1,
      "name": "HomeAway.com",
      "created_at": "2014-08-13T08:13:33Z",
      "updated_at": "2014-08-13T08:13:33Z"
    }
  ]
}
