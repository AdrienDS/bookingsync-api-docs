Photos

  1. List photos
  2. Get a single photo
  3. Create a new photo
  4. Update a photo
  5. Destroy a photo
  • OAuth Scopes:
  • public
  • rentals_write

Parameters

Name Type Read/Write Description
id Integer Read Photo’s id.
description Object Read Photo’s description, list of locales described in enums section.
kind String Read Photo’s kind, list of kinds described in enum section.
large_url String Read Photo’s large size url.
micro_url String Read Photo’s micro size url.
normal_url String Read Photo’s normal size url.
position Integer Read Photo’s position on the rental list.
thumb_url String Read Photo’s thumb size url.
created_at Time Read Photo’s create time.
updated_at Time Read Photo’s update time.

Parameters

Name Type Read/Write Description
id Integer Read Photo’s id.
description Object Read/Write Photo’s description, list of locales described in enums section. (80 characters max)
kind String Read/Write Photo’s kind, list of kinds described in enum section.
large_url String Read Photo’s large size url.
micro_url String Read Photo’s micro size url.
normal_url String Read Photo’s normal size url.
photo File Write Required. Photo’s image.
position Integer Read/Write Photo’s position on the rental list.
remote_photo_url String Write Photo’s remote url.
thumb_url String Read Photo’s thumb size url.
created_at Time Read Photo’s create time.
updated_at Time Read Photo’s update time.

Create a new photo

Creates a photo for given rental.

POST /rentals/:rental_id/photos
  • OAuth Scopes:
  • rentals_write

{

      "description": {
        "en": "Wonderful living room"
      },
      "kind": "livingroom",
      "photo": "livingroom.jpg",
      "position": 1
    }
  ]
}

Destroy a photo

Required OAuth scope: :rentals_write

Returns an empty response with ‘204 No Content’ status code on success.

DELETE /photos/:photo_id
