Create a new photo

Creates a photo for given rental.

POST /rentals/:rental_id/photos
Name Type Description
photo File Required. Photo’s image.
description String Photo’s description.
kind String Photo’s kind, list of kinds described in enum section.
position Integer Photo’s position.
  • OAuth Scopes:
  • rentals_write

Example JSON request

{
  "photos": [
    {
      "photo": "livingroom.jpg",
      "description": {
        "en": "Wonderful living room"
      },
      "kind": "livingroom",
      "position": 1
    }
  ]
}

Destroy a photo

Required OAuth scope: :rentals_write

Returns an empty response with ‘204 No Content’ status code on success.

DELETE /photos/:photo_id
